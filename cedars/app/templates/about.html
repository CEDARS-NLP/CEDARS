About

CEDARS is an open source software designed to aid researchers in quickly finding relevant events and their corresponding dates within large corpuses of medical notes.


Background

Developed out of a practical need for a more efficient way to conduct medical research, CEDARS aims to systematize and accelerate the aggregation of time-to-event data from the electronic health record (EHR). It accomplishes those goals by deploying natural language processing (NLP) as a tool to assist detection and characterization of clinical events by human annotators. The companion package PINES (Progressive Inference Networked Episodic Service) can be used to fit NLP models and deploy them within a CEDARS instance.


Documentation and Code

CEDARS and PINES are available as open-source Python packages. Each CEDARS project instance is deployed in a dedicated Docker container. The code base is maintained on GitHub. The latest stable Docker image can be downloaded here and the most recent development versions can be cloned from the respective repositories of CEDARS and PINES.

The annotator manual is available here.
